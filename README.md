# 🍽️ MealKart

MealKart is a full-stack food delivery web application developed using Django, Python, SQLite, HTML, CSS, and JavaScript.

## 🚀 Features

- User Registration and Login
- Secure Authentication
- Food Categories
- Restaurant Listings
- Food Search
- Add to Cart
- Quantity Management
- Checkout
- Multiple Payment Options
- Cash on Delivery
- Order Tracking
- Delivery Partner Details
- Admin Dashboard
- Responsive Design

## 🛠️ Technologies Used

- Python
- Django
- SQLite
- HTML5
- CSS3
- JavaScript

## ▶️ How to Run

```bash
python -m venv venv
```

Activate the virtual environment:

```bash
.\venv\Scripts\Activate
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Apply migrations:

```bash
python manage.py migrate
```

Run the server:

```bash
python manage.py runserver
```

Open your browser and visit:

```
http://127.0.0.1:8000/
```
