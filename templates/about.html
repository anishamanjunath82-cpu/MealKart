About mealKart

mealKart is a state-of-the-art full-stack food delivery application built to demonstrate modern web engineering and design principles. Inspired by premium global food applications like Zomato and Swiggy, mealKart integrates seamless frontend interactivity with a reliable Django backend database.

Our Core Technology Stack

  • Backend: Python 3.12, Django MVC framework, SQLite ORM.
  • Frontend: HTML5, Custom CSS3 grid layout, JavaScript Fetch API.
  • Design System: Premium glassmorphism design, responsiveness, custom loaders, and micro-animations.
  • Authentication: Django securely hashed passwords, remember sessions, and OTP simulations.

Our Features

We provide live search filters, restaurant review submissions, dynamic coupon validation, session-based cart updates, localized mock delivery tracking, and customizable user profiles with order history logs.
